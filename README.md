Bank Account Simulation

A simple and secure Flask-based banking simulation project built to demonstrate real-world banking operations.

About the Project

Bank Account Simulation is a web-based banking application developed using Python and Flask. The system allows users to perform essential banking operations such as account creation, login, deposits, withdrawals, balance enquiry, and viewing transaction history through a simple and user-friendly interface.

Features

* Create a new bank account
* Secure login and logout using sessions
* Deposit funds into an account
* Withdraw funds with balance validation
* Check account balance instantly
* View transaction history
* Handle invalid inputs and insufficient balance
* Interactive interface using HTML and CSS

Technologies Used

* Python 3
* Flask
* HTML5
* CSS3
* Jinja2 Templates

How to Run

1. Clone the repository

git clone [https://github.com/your-username/Bank-Account-Simulation.git](https://github.com/your-username/Bank-Account-Simulation.git)

2. Move to the project directory

cd Bank-Account-Simulation

3. Install Flask

pip install flask

4. Run the application

python server.py

5. Open your browser and visit

[http://127.0.0.1:5000](http://127.0.0.1:5000)
